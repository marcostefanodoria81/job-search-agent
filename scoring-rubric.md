# Job Scoring Rubric — Marco Doria

Usata dall'agente per valutare ogni offerta e restituire un match score (0–100) + gap analysis sintetica.

---

## Profilo di riferimento

- **Ruolo attuale:** Affiliate Content Coordinator → target Editorial Manager / Content Lead
- **Esperienza:** 8+ anni (traduzione EN/IT 2013–2021, Senior Editor 2019–2023, team lead 2023–oggi)
- **Competenze core:** editorial operations (hiring, QA, onboarding, pipeline), affiliate content, SEO editoriale, GA4 (certificato), Awin, WordPress
- **Lingue:** Italiano (madrelingua), Inglese C2
- **Posizione:** Sicilia — fully remote obbligatorio
- **Range target:** €40.000–€60.000 lordi (o equivalente in USD/GBP)
- **Settori preferiti:** eCommerce, SaaS, Tech, Media/Publishing
- **Settori da escludere:** gaming, crypto/Web3
- **Settori bassa priorità:** healthcare, legal, finance

---

## Criteri ELIMINATORI (se fallisce uno, score massimo = 30)

| Criterio | Condizione eliminatoria |
|----------|------------------------|
| Remote | Il ruolo NON è fully remote (on-site o ibrido obbligatorio) |
| Timezone | Richiede overlap esclusivo con UTC-8 / UTC-7 (West Coast US) — Sicilia UTC+2 incompatibile |
| Settore | Gaming, crypto/Web3 |
| Livello | Entry level o internship |
| Salario dichiarato | Sotto €30.000 / $33.000 equivalenti |
| Lingua | Richiede madrelingua in lingua diversa da italiano o inglese |

## Dimensioni di scoring (totale 100 punti)

### 1. Fit ruolo (30 punti)

| Scenario | Punti |
|----------|-------|
| Editorial manager / content lead / head of content con team leadership | 30 |
| Content manager con ownership strategica (no team) | 22 |
| Content marketing manager con focus SEO/performance | 18 |
| Affiliate manager con componente editoriale | 15 |
| Senior editor / writer con prospettiva di crescita | 12 |
| Pure copywriter / social media manager | 5 |
| Ruolo tecnico, sales, developer relations | 0 |

### 2. Fit settore (20 punti)

| Scenario | Punti |
|----------|-------|
| eCommerce o media/publishing con affiliate | 20 |
| SaaS / Tech | 16 |
| Media digitale / news / publishing generalista | 14 |
| FinTech (no crypto) | 10 |
| Healthcare, legal, education | 6 |
| Gaming, crypto/Web3 | 0 (+ flag eliminatorio) |

### 3. Fit salariale (20 punti)

| Scenario | Punti |
|----------|-------|
| €50.000+ dichiarato (o $55k+) | 20 |
| €40.000–€49.999 (o $44k–$54k) | 14 |
| Salario non dichiarato | 10 (neutro, non penalizzare) |
| €30.000–€39.999 | 4 |
| Sotto €30.000 | 0 (+ flag eliminatorio) |

### 4. Skill match (20 punti)

Valutare quante delle seguenti skill Marco copre vs quelle richieste dal JD:

| Skill richiesta dal JD | Copertura Marco |
|------------------------|-----------------|
| Editorial planning / content calendar | Forte |
| Team management / editorial leadership | Forte |
| SEO editoriale (buying guides, keyword research) | Forte |
| GA4 / Google Analytics | Forte (certificato) |
| Affiliate content / performance-driven content | Forte |
| Awin o altri affiliate network | Medio (solo Awin dichiarato) |
| WordPress / CMS | Medio |
| B2B SaaS content marketing | Debole (gap) |
| Data journalism / insights-driven content | Debole (gap) |
| Video / podcast content | Assente |
| PR / media outreach | Assente |

**Calcolo:** (skill coperte bene × 2 + skill coperte parzialmente × 1) / (skill richieste totali × 2) × 20

### 5. Colmabilità dei gap (10 punti)

Valutare quanto velocemente Marco potrebbe coprire i gap rilevanti del JD lavorando con un AI collaborator (cover letter su misura, portfolio piece ad hoc, preparazione colloquio).

| Gap rilevato | Colmabilità | Logica |
|--------------|-------------|--------|
| B2B SaaS content | Alta | Stesso meccanismo del contenuto affiliate performance-driven — si traduce in linguaggio |
| GEO / AI search | Alta | Framework concettuale acquisibile rapidamente; Marco conosce già il problema da utente |
| Portfolio pubblico in inglese | Alta | The Gentle Meeple esiste già come writing sample |
| Data journalism / insights | Media | Richiede un mini case study concreto su dati GA4/SEOZoom — costruibile |
| Affiliate network aggiuntivi | Media | Dichiarabili solo con esperienza reale, nessun shortcut |
| Video / podcast content | Bassa | Non simulabile senza pratica diretta |
| Team management enterprise | Bassa | Richiede contesto organizzativo che Marco non ha avuto |

**Calcolo:** se i gap critici del JD hanno colmabilità Alta o Media, assegnare 8–10 punti. Se i gap sono Bassi o strutturali, assegnare 0–4.

| Scenario | Punti |
|----------|-------|
| Gap principali tutti colmabili (Alta/Media) con supporto AI | 10 |
| Gap misti: alcuni colmabili, uno strutturale | 6 |
| Gap principali a colmabilità Bassa o richiedono anni di pratica | 2 |
| Nessun gap rilevante | 10 (bonus pieno) |

## Note operative

- Se il salario non è dichiarato, non penalizzare — molte company EU non lo pubblicano
- Se la location è "worldwide" senza restrizioni, considerare come remote-friendly
- Timezone UTC+2 (Sicilia) è compatibile con: USA East Coast overlap parziale, tutta Europa, UK, Africa
- Non usare gaming come settore target anche se il ruolo è senior e ben pagato
- La cover letter va personalizzata solo per score ≥ 65
